Plot-System Bauwelt

In der Bauwelt nutzt der Server ein Plot-System, in dem jeder Spieler durch Plots (bauflächen) eigene Bereiche verwalten kann.

Plots (Grundstücke):
Plots sind rechteckige Flächen, die sich aneinandergereiht in einer Rasterwelt befinden. Üblicherweise muss man einen Plot belegen (claimen), um darin zu bauen.

Wichtige Befehle:
```
/plot claim            // Claimt den Plot, auf dem du gerade stehst.
/plot auto             // Automatisch nächst freien Plot claimen.
/plot home             // Teleportiert dich zu deinem ersten Plot.
/plot visit <Spieler>  // Teleportiert dich zum Plot eines anderen Spielers.
/plot add <Spieler>    // Füge einen Co-Besitzer für deinen Plot hinzu.
/plot remove <Spieler> // Entfernt einen Co-Besitzer.
/plot list             // Zeigt eine Übersicht deiner Plots im Chat-GUI.

```

(Hinweis: Die Befehle können je nach Server variieren. Das sind allgemeine Beispiele.)

Beispiel:

/plot claim – Du erhältst automatisch den Plot unter dir, falls dieser frei ist.

/plot auto – Du bekommst den nächsten freien Plot in der Welt.

/plot home – Teleportiert dich zum Grundstück, das du bereits besessen hast (deinem Plot).

/plot visit Spieler123 – Teleportiert dich zum Plot von „Spieler123“ (falls du ihn kennst oder ein Freund bist).

GUI/Ingame-Interaktionen:

Häufig öffnet sich beim /plot list ein GUI-Menü mit deinen Plots (kleine Karten oder Namen).

Plots sind meist von Zäunen oder Mauern umgeben, sodass du den eingekreisten Bereich gut erkennst.

Serverunterschiede:

Nur Bauwelt: Das Plot-System gilt nur in der Bauwelt. Im Adventure gibt es kein Plot-System.

Ändert sich oft je nach Gebiet: Es können mehrere Plotgrößen existieren (z.B. StarterPlots, Inselplots).

Tipps / Best Practices:

Claim sofort einen Plot mit /plot claim, sonst könnte ein anderer Spieler ihn nehmen.

Nutze /plot list oder schau dich um, um Nachbarplots zu entdecken.

Vergib eigene Namen oder benutze /plot sethome, wenn unterstützt, damit du leicht zu deinem Plot kommst.

Einschränkungen / Fallstricke:

Man kann normalerweise nur Claims innerhalb eines Plots machen. In Bauwelt reicht ein Plot pro Spieler.

Aufbau außerhalb deines Plots (auch Zäune/Leitern, etc.) wird nicht geschützt – bleibe in deinen Grenzen, um Ärger zu vermeiden.

FAQ:

F: Wie viele Plots darf ich haben?
A: Standardmäßig darf jeder Spieler einen Plot claimen.

F: Kann ich bei einem anderen Plot mitleveln oder dazu-kaufen?
A: Nein, ohne Freigabe des Besitzers kannst du keinen Plot verändern. Mehr Plot-Slots gibt es nur durch Gruppen oder Rang.
